# tts_box - EN
Multi-Language TTS Box (Portable Version)
A lightweight, professional text-to-speech utility with custom pause control.

- Features
Portable App: No installation required. Simply run the .exe file to start using it. It won't mess with your system registry.
Multilingual Support: Supports Chinese, English, Japanese, Malay, Korean, Thai, Vietnamese, and Tamil. （Only Simplified Chinese)
Dynamic UI: Interface labels and hints automatically adapt to the language you select.
Pause Customization: Precisely control the silence duration (in seconds) between each line of text.
Self-Cleaning Cache: All temporary audio fragments are managed within the Windows Temp folder and deleted upon exit.

- Quick Start
1. Select Language: Choose the target language from the top dropdown menu.
2. Input Text: Paste your text into the main editor. Each new line represents a segment followed by a pause.
3. Adjust Interval: Enter the desired pause length (e.g., 1.0 for 1 second) in the input field.
4. Actions:
   - Click Preview: Listen to the generated speech immediately.
   - Click Save MP3: Export and save your audio to a local folder.

- Requirements
Active internet connection required for Neural TTS synthesis.
Compatible with Windows 10/11.
FFmpeg must be installed on your system.
1. Get the FFmpeg essentials from [FFmpeg.org](https://ffmpeg.org/download.html).
2. Unzip the folder to a permanent path (e.g., `C:\ffmpeg`).
3. Edit your System Environment Variables and add the `bin` folder path to `Path`.
4. Run `ffmpeg -version` in CMD to confirm.

# tts_box - CN
多國語言語音合成盒 (Portable 免安裝版)
這是一個輕量、專業的文字轉語音工具，支持自定義停頓秒數。

- 軟體特性
免安裝版 (Portable)：本軟體為綠色免安裝版，下載後直接點擊 .exe 即可使用，不會在您的系統中留下冗餘路徑。
多語言支持：支援中、英、日、馬來、韓、泰、越、淡米爾語。（中文目前只有簡體）
智能介面聯動：介面上的按鈕與標籤會隨選擇的語言自動切換（例如切換至泰語時，所有提示將變更為泰文）。
精準停頓控制：可設置每行文字之間的停頓秒數，讓朗讀節奏更符合您的需求。
乾淨的系統管理：所有暫存音頻都會在 Windows 系統 Temp 資料夾中處理，並在程式關閉時自動清理。

- 如何使用

1. 選擇語言：從頂部的選單中選擇您要合成的語言。
2. 輸入文字：在主視窗貼上文字。注意：每個需要停頓的地方請進行「換行」。
3. 設定秒數：在下方填寫每段之間的停頓時間（例如 1.5 秒）。
4. 操作：
   點擊 試聽：立即播放合成效果。
      點擊 保存 MP3：選擇位置並匯出最終的音頻檔案。

- 系統需求
需在有網路連接的 Windows 10/11 環境下執行。
您的系統必須安裝FFmpeg。
1. 前往 [FFmpeg 官網](https://ffmpeg.org/download.html) 下載 Windows 版本。
2. 將資料夾移動到固定路徑（例如 `C:\ffmpeg`）。
3. 在系統的「環境變數」中，將 `bin` 資料夾的路徑加入到 `Path`。
4. 在命令提示字元輸入 `ffmpeg -version`，看到版本資訊即代表成功。

# tts_box - BM
Kotak TTS Pelbagai Bahasa (Versi Portable)
Alat pertukaran teks-ke-ucapan (TTS) yang ringan dan profesional dengan fungsi jeda tersuai.

- Ciri-ciri Utama
Versi Portable: Tidak perlu dipasang. Hanya jalankan fail .exe terus dari folder anda. Ia tidak akan meninggalkan sebarang fail sampah dalam sistem registry anda.
Sokongan Pelbagai Bahasa: Menyokong Bahasa Melayu, Cina, Inggeris, Jepun, Korea, Thai, Vietnam, dan Tamil.
Antara Muka Dinamik: Label dan arahan pada aplikasi akan bertukar secara automatik mengikut bahasa yang anda pilih.
Kawalan Jeda (Pause) Tepat: Anda boleh menetapkan durasi jeda (dalam saat) antara setiap baris teks mengikut keperluan anda.
Pembersihan Automatik: Semua fail audio sementara diuruskan di dalam folder Windows Temp dan akan dipadamkan secara automatik apabila aplikasi ditutup.

- Cara Penggunaan
1. Pilih Bahasa: Pilih bahasa sasaran daripada menu lungsur (dropdown) di bahagian atas.
2. Masukkan Teks: Tampal teks anda dalam kotak utama. Nota: Pastikan setiap ayat yang memerlukan jeda berada di baris baharu.
3. Tetapkan Jeda: Masukkan tempoh masa jeda (contohnya 1.0 untuk 1 saat) pada ruangan yang disediakan.
4. Tindakan:
   - Klik Dengar: Untuk mendengar hasil suara dengan serta-merta.
   - Klik Simpan MP3: Untuk menyimpan hasil akhir ke dalam komputer anda.

- Keperluan Sistem
Memerlukan sambungan internet untuk sintesis suara Neural TTS yang berkualiti tinggi.
Sesuai untuk Windows 10 dan Windows 11.
FFmpeg mesti dipasang pada sistem anda.
1. Layari [FFmpeg.org](https://ffmpeg.org/download.html) untuk memuat turun fail Windows.
2. Nyahzip fail ke lokasi kekal (contohnya `C:\ffmpeg`).
3. Edit "Environment Variables" sistem dan masukkan laluan folder `bin` ke dalam `Path`.
4. Taip `ffmpeg -version` dalam Terminal/CMD untuk memastikan ia berfungsi.
